Mac OS Tile Wallpaper Recreations

These are wallpaper tiles I created. They are based on the embossed Finder tiled wallpaper images from Mac OS 7.5 and Mac OS 8.5, except they use the newer Finder icon from macOS 11 and the newer SF font. Also included is a recreation of the desktop tile used on Mac OS 8/9 CD installers (when booted as a startup disk).

Included are 128px and 256px tiled wallpapers images. Keep in mind, classic Mac OS tiles each image from the top left corner, while modern macOS centers the pattern.

Big Sur Icon Packs

These are macOS Big Sur style icons I've created. Included here are icon packs for various emulator apps including Mini vMac, LisaEm, Dolphin, and more! Just download the zip file for each pack.

To add the icon to an application, select the application in the finder. Go to the file menu and click "Get Info" to open a window containing information about the applicaton file. Next, find the PNG of the icon you wish to add and drag it to the app icon in the info window.

For help installing the icons, see this page on macosicons.com. To use the disk image icons with Mini vMac, use the instructions on this page on the Mini vMac website.
